Course Overview

This course provides foundational knowledge in fonts, encoding, and script display for language technology work. You'll learn to recognize, understand, and diagnose common text display problems that occur when working with multilingual data, particularly IPA transcriptions and minority language scripts.

By the end of this course, you will be able to:

  • Recognize patterns in broken text (boxes, wrong marks, mojibake)
  • Understand how fonts, encodings, and rendering work together
  • Diagnose font and character display issues systematically
  • Use appropriate tools to investigate encoding and font problems
  • Recommend suitable fonts for linguistic work
  • Explain problems clearly to colleagues and collaborators

Course Modules

Module 1A: Why Does Text Break?

45-60 minutes

Focus: Pattern recognition and observation

Learn to identify three common patterns of broken text: boxes (missing glyphs), wrong marks (combining character issues), and mojibake (encoding mismatches). Build the foundation for diagnostic work.

Key Activities:

  • Recognize display problem patterns
  • Practice identifying problem types
  • Apply pattern recognition to real scenarios

Module 1B: The Three Culprits

45-60 minutes

Focus: Understanding how text display works

Dive deep into the three systems that must work together for text to display correctly: encoding (the map), fonts (the shapes), and rendering (the assembly). Understand why each type of problem occurs.

Key Activities:

  • Explore encoding systems (Unicode, legacy encodings)
  • Compare font coverage across different fonts
  • Investigate combining characters and normalization
  • Practice diagnosing multiple concurrent problems

Module 1C: Your Diagnostic Toolkit

45-60 minutes

Focus: Practical diagnostic skills and resources

Build your practical toolkit for investigating text problems. Learn to check file encodings, verify font coverage, identify Unicode values, and access essential resources. Complete a comprehensive diagnostic challenge.

Key Activities:

  • Use tools to inspect file encoding
  • Check font coverage systematically
  • Identify Unicode code points
  • Build a personal reference guide
  • Complete multi-problem diagnostic scenario

Prerequisites

This course assumes:

  • Basic computer literacy (opening files, using web browsers)
  • Some exposure to multilingual text or linguistic data
  • Access to a computer with internet connection
  • Ability to install fonts and software (optional but helpful)

No prior knowledge of encodings, Unicode, or font technology is required.
